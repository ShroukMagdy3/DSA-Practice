The goal is to improve problem-solving skills and implement common algorithms.

## 📂 Day 1 – Frequency Counters, Multiple Pointers, Sliding Window, Longest Substring

### 1️⃣ Frequency Counter Patterns
- **`same(arr1, arr2)`** – checks if one array has the squares of another
- **`validAnagram(str1, str2)`** – checks if two strings are anagrams

### 2️⃣ Multiple Pointers
- **`sumZero(sortedArr)`** – finds the first pair in a sorted array whose sum is 0
- **`countUniqueValues(sortedArr)`** – counts unique values in a sorted array

### 3️⃣ Sliding Window
- **`maxSubArr(arr, num)`** – finds the maximum sum of `num` consecutive elements
- **`minSubArrayLen(arr, target)`** – finds the minimal length of a subarray with sum ≥ target

### 4️⃣ Longest Substring
- **`findLongestSubstring(str)`** – finds the length of the longest substring without repeating characters
